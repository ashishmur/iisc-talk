## 2C. The Economic Equation: Production Functions

### Classic Factors of Production

The classical formulation lists four factors:
1. **Land** (natural resources)
2. **Labor** (human work)
3. **Capital** (machinery, tools, financial capital)
4. **Technology/Entrepreneurship** (knowledge, innovation, organization)

### The Standard Cobb-Douglas Production Function

```
Y = A * L^alpha * K^beta
```

Where:
- **Y** = Total output (GDP)
- **A** = Total Factor Productivity (TFP), representing technology, efficiency, knowledge
- **L** = Labor input
- **K** = Capital input
- **alpha** = Output elasticity of labor (typically ~0.6-0.7)
- **beta** = Output elasticity of capital (typically ~0.3-0.4)
- If alpha + beta = 1, constant returns to scale

**For the slide, the simplified version:**
```
Output = Technology x Labor^0.7 x Capital^0.3
```

### How AI Changes the Equation: Acemoglu's Task-Based Framework

Daron Acemoglu (MIT, Nobel Laureate 2024) provides the most rigorous framework:

**Traditional view:** Technology (A) augments both labor and capital.

**Acemoglu's task model:** Production requires a set of tasks. Tasks can be allocated to either labor or capital. AI = expanding the set of tasks performed by capital (automation).

**The Two Competing Effects:**
1. **Displacement effect:** AI automates tasks previously done by humans, reducing labor demand
2. **Reinstatement effect:** New tasks emerge where humans have comparative advantage, increasing labor demand

**Acemoglu's Empirical Estimates (2024 paper: "The Simple Macroeconomics of AI"):**
- ~20% of U.S. labor tasks are exposed to AI
- Only ~23% of those exposed tasks can be profitably automated (given current costs)
- Average labor cost savings per automated task: ~27%
- Using Hulten's theorem: total TFP impact = 0.20 x 0.23 x 0.27 = ~0.53-0.66% over 10 years
- That is modest: roughly 0.05-0.07% per year added to TFP growth

**For the slide, the new equation:**
```
BEFORE:  Y = A * L^0.7 * K^0.3

AFTER:   Y = A * (H + AI)^alpha * K^beta

Where H = human labor on remaining tasks
      AI = automated task output (capital performing former labor tasks)
```

**The key insight:** Technology (A) is no longer just a multiplier sitting outside the equation. AI is actively substituting inside the labor term, converting L tasks into K tasks. The boundary between labor and capital is dissolving.

**Task displacement explained 50-70% of wage structure changes from 1980-2016.**

### The Solow Paradox Returns (February 2026)

Robert Solow, 1987: "You can see the computer age everywhere but in the productivity statistics."

Erik Brynjolfsson (Stanford): General-purpose technologies require complementary investments in organizational redesign, worker training, and process reengineering before benefits materialize. This "J-curve" effect means productivity initially dips before it rises.

The February 2026 NBER study: 90% of firms see zero productivity impact from AI. Firms forecast only +1.4% TFP and +0.8% output over the next 3 years. Exactly the pattern Solow described 39 years ago.

## Key Narrative Threads for the Presentation

### Thread 1: "Every revolution follows the same script"
Installation (hype, speculation, bubble) then Crash then Deployment (golden age, real value)

### Thread 2: "The crash is not the end, it's the turning point"
Canal Mania led to industrial golden age. Railway Mania led to global trade. 1929 led to the greatest prosperity boom in history. Dotcom led to Google, Amazon, Facebook. The AI reckoning of 2025-2026 is not the death of AI, it's the birth of the deployment phase.

### Thread 3: "Each wave eats more of the labor stack"
- Wave 1-2: Physical labor displaced (agriculture 70% to 10%)
- Wave 3-4: Operational labor displaced (manufacturing peaked, declined)
- Wave 5: Information work transformed (services sector digitized)
- Wave 6: Intellectual labor threatened (the last frontier)

### Thread 4: "The production function is being rewritten"
Technology used to multiply human effort. Now it substitutes for it. The question is not "will AI increase productivity?" but "will humans remain in the equation at all?"
